MDI

EL sistema hotel san Carlos presenta un sistema unificado para el control de las actividades comerciales del hotel tanto en la parte externa, como el manejo se sus clientes como de todos los procesos asociados a su funcionamiento.


Opciones

Dentro de su barra principal encontramos las siguientes opciones que le permitirán a los usuarios manejar las actividades del Hotel, cada una de estas opciones nos llevará a las diferentes ventanas respectivas a cada uno de los módulos.

Finanzas

Dentro del modulo de finanzas encontraremos las opciones para el control de la contabilidad de la empresa, así como de las actividades bancarias en la subpestaña de bancos.

Compras

Dentro del modulo de compras encontraremos las opciones para navegar dentro de las vistas de manejo para las opciones de compras y de las cuentas por pagar.

Ventas

Dentro del módulo de ventas encontraremos las opciones para navegar dentro de las vistas de manejo para las opciones de ventas y de las cuentas por cobrar.

Herramienta

Dentro de las opciones de las herramientas, encontramos las opciones para realizar los manejos de la seguridad, reportes y ayudas del sistema.
